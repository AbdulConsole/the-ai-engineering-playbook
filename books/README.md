# Books

A curated list of books for software engineers learning AI-assisted engineering, AI coding workflows, agentic development, and modern software engineering with AI.

---

# AI-Assisted Software Engineering

## 1. *Coding with AI* — Jeremy Morgan

A practical introduction to using AI tools in day-to-day software development workflows, including AI pair programming and productivity enhancement.

* Website: [Jeremy Morgan Books](https://www.jeremymorgan.com/books/?utm_source=chatgpt.com)

---

## 2. *AI-Powered Developer* — Nathan B. Crocker

Focuses on how developers can integrate generative AI into modern engineering workflows, tooling, and software delivery practices.

* Official page: [AI-Powered Developer (Manning)](https://www.manning.com/books/ai-powered-developer?utm_source=chatgpt.com)
* Publisher page: [Simon & Schuster Listing](https://www.simonandschuster.com/books/AI-Powered-Developer/Nathan-B-Crocker/9781633437616?utm_source=chatgpt.com)
* O’Reilly learning page: [AI-Powered Developer on O’Reilly](https://www.oreilly.com/library/view/ai-powered-developer/9781633437616/?utm_source=chatgpt.com)

([Manning Publications][1])

---

## 3. *AI-Assisted Coding* — Michael Kofler, Bernd Öggl, Sebastian Springer

A comprehensive guide to AI pair programming, prompt engineering, debugging, refactoring, and modern AI coding workflows.

* Official publisher page: [AI-Assisted Coding (Rheinwerk)](https://www.rheinwerk-verlag.de/ai-assisted-coding-practical-guide-for-software-development/?utm_source=chatgpt.com)
* SAP Press listing: [AI-Assisted Coding (SAP Press)](https://www.sap-press.com/ai-assisted-coding_6058/?utm_source=chatgpt.com)
* Author information: [Authors & Backgrounds](https://www.rheinwerk-verlag.de/ai-assisted-coding-practical-guide-for-software-development/autoren/?utm_source=chatgpt.com)

([Rheinwerk Verlag][2])

---

## 4. *AI-Assisted Programming* — Tom Taulli

Covers the fundamentals of programming with AI assistance, including practical use cases and workflow strategies.

* Publisher page: [Tom Taulli Books](https://www.apress.com/gp/search?query=Tom+Taulli&utm_source=chatgpt.com)

---

## 5. *Generative AI for Software Development* — Sergio Pereira

A guide to applying generative AI tools and large language models throughout the software development lifecycle.

* Publisher page: [Packt Publishing – Sergio Pereira Books](https://www.packtpub.com/en-us/search?query=Sergio+Pereira&utm_source=chatgpt.com)
