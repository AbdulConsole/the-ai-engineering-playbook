# Courses

A curated list of courses for software engineers learning AI-assisted development, AI coding workflows, prompt engineering, agentic systems, and modern software engineering with AI.

# Recommended Courses

## 1. IBM — Generative AI for Software Developers

One of the strongest beginner-to-intermediate tracks focused specifically on software engineering workflows with generative AI.

Topics include:

* Prompt engineering

* AI-assisted coding

* GitHub Copilot

* ChatGPT workflows

* AI-powered testing

* AI software engineering practices

* Course: [Generative AI for Software Developers (Coursera)](https://www.coursera.org/specializations/generative-ai-for-software-developers?utm_source=chatgpt.com) ([Coursera][1])

---

## 2. DeepLearning.AI — Generative AI for Software Development

Taught by Laurence Moroney (former AI lead at Google), this course focuses on integrating generative AI into real-world development workflows.

Topics include:

* AI-assisted coding

* Debugging with AI

* Documentation generation

* Testing workflows

* Development productivity

* Course: [Generative AI for Software Development](https://www.deeplearning.ai/courses/generative-ai-for-software-development/?utm_source=chatgpt.com) ([deeplearning.ai][2])

---

## 3. AWS — Generative AI for Developers

A practical AWS-focused course covering deployment and integration of generative AI systems into production applications.

Topics include:

* Prompt engineering

* GenAI deployment

* AI application architecture

* Building AI-powered software

* Course: [AWS Generative AI for Developers](https://www.futurelearn.com/courses/generative-ai-for-developers?utm_source=chatgpt.com) ([FutureLearn][3])

---

## 4. GenAI for Software Developers Specialization

A practical specialization focused on integrating tools like ChatGPT and GitHub Copilot into software engineering workflows.

Topics include:

* AI-assisted development

* AI code review

* Full-stack AI workflows

* Debugging with AI

* Prompt engineering

* Software modernization

* Course: [GenAI for Software Developers Specialization](https://www.coursera.org/specializations/generative-ai--for-software-developers?utm_source=chatgpt.com) ([Coursera][4])
